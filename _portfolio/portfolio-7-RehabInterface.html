This project is a part of the research on Rehabilitation Robotics at ARMLab, Isfahan University of Technology, Iran. The aim was to develop an interactive interface between the KUKA robot and stroke/SCI patients, which brings the human in the control loop of the robot and thus triggers the neuroplasticity in the process of arm-hand rehabilitation.

By using a communication method called “Named Pipe” the interface (written in C#) can reliably communicate with the KUKA controller program (written in C++) within a separate thread, which guarantees an uninterrupted operation.
